Browser Flocking Simulator

An HTML5 Canvas and JavaScript implementation of Craig Reynolds' boids algorithm.

View on GitHub Live Demo

An early implementation of flocking behavior (boids). This was a first introduction to 2D graphics and emergent AI behavior.

The current implementation demonstrates separation, alignment, and cohesion forces applied to multiple species of boids. While functional, the system has known limitations that a future rewrite would address.

Planned improvements include replacing the internal data model with proper position/direction vectors, applying per-species coefficients for behavior tuning, implementing a quadtree for O(n log n) neighbor lookups instead of the current O(n²) approach, and potentially moving to 3D using WebGL or OpenGL compute shaders.
