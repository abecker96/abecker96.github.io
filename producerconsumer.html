Multithreaded Producer-Consumer Solution

A C++ project written for my Operating Systems class.

An arbitrary number of producer threads place random number in a bounded buffer, and an arbitrary number of consumer threads remove items from the buffer, determining if the numbers are prime. Written in C++ for unix systems.
